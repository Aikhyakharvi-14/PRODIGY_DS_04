# PRODIGY_DS_04
This project was completed as part of the Prodigy InfoTech Data Science Internship program. The objective of the project is to analyze and visualize sentiment patterns in social media data to better understand public opinions and attitudes toward different topics or brands.
Using Twitter training and validation datasets, the project focuses on identifying sentiment categories such as Positive, Negative, Neutral, and Irrelevant. The data was cleaned, processed, analyzed, and visualized to extract meaningful insights from social media conversations.

Objectives
* Analyze sentiment distribution in social media datasets
* Perform data cleaning and preprocessing
* Compare sentiment patterns between datasets
* Create meaningful visualizations for better understanding
* Understand public opinion trends using data analysis techniques

 Technologies Used
* Python
* Google Colab
* Pandas
* Matplotlib

 Project Workflow
 1. Data Collection
Imported Twitter training and validation datasets provided for the internship task.

 2. Data Cleaning
* Removed missing values
* Checked dataset structure
* Assigned proper column names

3. Sentiment Analysis
Analyzed sentiment categories including:
* Positive
* Negative
* Neutral
* Irrelevant

 4. Data Visualization
Created comparative bar charts to visualize sentiment distributions between training and validation datasets.

Key Insights
* Positive and negative sentiments were the most frequently occurring categories.
* Training and validation datasets showed similar sentiment distribution patterns.
* Visualization helped identify public opinion trends effectively.

Skills Gained

Conclusion
This project provided hands-on experience in analyzing real-world social media data and understanding how sentiment analysis can be used to interpret public opinion and user attitudes using Python-based data analytics techniques.
